About Me

Hi! I'm Jan Michael Velasco, a passionate and skilled WordPress developer with over 15 years of experience building responsive, user-friendly websites. Throughout my career, I've specialized in custom WordPress theme and plugin development, delivering tailored solutions that combine performance, functionality, and design.

My journey began in 2008 and has taken me through various companies and industries, including relocation services, construction, and digital marketing. I've had the opportunity to collaborate with designers, marketers, and developers to build everything from eCommerce platforms to lead-generation tools.

Whether I'm optimizing SEO, solving complex bugs, or enhancing user experience, I bring a hands-on, detail-oriented approach to every project. I'm also comfortable with version control tools like Git, and I stay up to date with the latest web technologies to ensure every build is modern, mobile-optimized, and aligned with UX best practices.

Some notable brands I've contributed to include:

  • Santa Fe Relocation
  • Polyfabrics
  • Fortress Fencing
  • Signfix Calculator

When I'm not coding, you'll likely find me exploring new tech, refining my CSS animations, or assisting businesses in leveling up their digital presence.

Skills & Technologies

Strong understanding of semantic HTML5, modern CSS (Flexbox, Grid), responsive design principles, and accessibility best practices.

Proficient in vanilla JavaScript and ES6+, including DOM manipulation, event handling, and API interactions.

Experienced in building reusable components, using hooks, and managing state with React and related libraries.

Comfortable with theme customization, page builders, custom post types, and template hierarchy in WordPress.

Daily usage of Git for source control, with knowledge of branching strategies, pull requests, and GitHub workflows.
